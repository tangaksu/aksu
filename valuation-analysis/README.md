## 功能介绍

本技能提供一套完整的价值投资分析体系，基于晨星公司首席股票分析师帕特·多尔西的经典方法论，帮助你从护城河、财务健康、管理层、成长性、估值五个维度全面评估一家公司的投资价值。

### 核心能力

| 功能模块 | 说明 |
|---------|------|
| **护城河分析** | 识别公司可持续竞争优势（品牌、成本优势、网络效应、转换成本等） |
| **财务健康检查** | 六项危险信号排查，识别财务造假/风险 |
| **成长性分析** | 增长来源分解 + 行业生命周期定位 |
| **管理层评估** | 能力/诚信/股东导向三维评估框架 |
| **DCF估值** | 晨星标准现金流折现模型，计算内在价值和安全边际 |
| **同业对比** | 横向估值比较，判断相对位置 |
| **投资决策整合** | 综合评分卡 + 买入/卖出决策矩阵 |

### 输出报告

标准分析报告包含以下章节：

1. **概述（执行摘要）** - 投资评级、目标价、安全边际、一句话结论
2. **十分钟测试** - 快速筛选9个问题
3. **护城河分析** - 四步检验法
4. **财务健康检查** - 危险信号排查 + 健康指标
5. **成长性分析** - 增长来源 + 成长性指标
6. **管理层评估** - 能力/诚信/股东导向
7. **估值分析（DCF）** - 五步法内在价值计算
8. **同业对比估值** - 多方法估值对比
9. **投资决策整合** - 综合评分 + 操作建议

## 使用方式

### 用户需要提供

| 数据类型 | 是否必须 | 说明 |
|---------|---------|------|
| **股票代码** | 必须 | A股（如 300750）或港股（如 00700） |
| **最新财报** | 强烈推荐 | 年度报告/中期报告PDF，评估将优先基于财报数据 |
| **当前股价** | 可选 | 用于计算安全边际，系统也会自动获取 |

### 分析流程

```
用户提供股票代码（+ 财报PDF）
    ↓
系统提取财务数据
    ↓
执行8章完整分析
    ↓
生成PDF报告
```

### 示例使用

```
用户：分析一下宁德时代

系统：请上传宁德时代最新年度报告PDF，我将基于财报数据进行完整价值投资分析。

[用户上传PDF]

系统：分析报告生成完成...
```

## 核心阈值参考

| 指标 | 优秀标准 | 一般 | 危险 |
|------|---------|------|------|
| ROE | >15%连续5年+ | 10-15% | <10% |
| FCF/销售额 | >5% | 0-5% | <0% |
| 毛利率 | >40% | 20-40% | <20% |
| 负债/权益 | <1.0 | 1.0-1.5 | >1.5 |
| 安全边际 | >20% | 10-20% | <10% |

## 免责声明

本技能基于《股市真规则》价值投资框架及公开财务数据进行分析，仅供参考，**不构成投资建议**。投资者应独立判断并承担投资风险。

历史业绩不代表未来表现，估值模型对假设高度敏感，请结合多维度信息做出投资决策。
