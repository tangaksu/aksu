## 🎯 入场决策模板(强制使用)

每次买入建议 必须 包含以下 8 项:

```
📊 标的: XXX

📊 入场计划:
  入场价: ¥X
  仓位: Y%(仓位上限 50%)

📊 价格空间分析:
  上涨空间: ¥A (+B%)
  下跌空间: ¥C (-D%)
  风险收益比: B/D = E:1
  ⚠️ 风险收益比 < 2 = 不入场

📊 出场计划(一次写完,入场后不增减):
  止盈 1: ¥X1 → 卖 N1 股
  止盈 2: ¥X2 → 卖 N2 股
  止盈 3: ¥X3 → 卖 N3 股
  止损 1: ¥Y1 → 卖 M1 股
  止损 2: ¥Y2 → 卖 M2 股(全清)

📊 仓位检查:
  □ 单股 ≤ 50%
  □ 持仓总数 ≤ 5 只
  □ 是否触发总资金红线
  □ 是否需要先减仓其他

📊 资金来源检查:
  □ 现金还是融资?
  □ 如融资 → 是否符合"正期望策略验证"

📊 时效核查:
  □ 关键数据/消息 日期
  □ 是否已 price-in
  □ 是否有新催化
```

## 🚨 情绪化交易拦截器

### 4 类必须拦截的心态

#### 心态 1: FOMO (Fear of Missing Out)
```
表现:
  • "再升涨 +123% 我要追"
  • "天岳又涨了 后悔卖"

拦截信号:
  • 想买 已涨 > 30% 的票
  • 想追 涨停 / 一字板

处理:
  → 暂停 24 小时
  → 加入盯盘 等回调 -10% 再入
```

#### 心态 2: 反思过度补偿
```
表现:
  • "上次卖少了 这次要多卖"
  • "5/27 我太保守"

拦截信号:
  • 减仓 > 入场计划的 50%
  • 因为"前一次错过"而 加大此次操作

处理:
  → 严格按 入场计划 执行
  → 不因为 后悔 改变 当前规则
```

#### 心态 3: 报复性交易
```
表现:
  • "刚亏了 要赶紧赚回"
  • "新莱亏 ¥3 万 用别的票补回"

拦截信号:
  • 24h 内 亏损后 立刻 新建仓
  • 增加 高弹性 高风险 标的

处理:
  → 强制 24h 冷静期
  → 不在亏损后 立刻 新决策
```

#### 心态 4: 抄底心态
```
表现:
  • "便宜了就买"
  • "跌破均线就抄"
  • "PE 比同行低就买"

拦截信号:
  • 仅凭"便宜" 入场
  • 没有 底部确认 信号

处理:
  → 必须有 多重 底部信号 (缩量+企稳+板块转)
  → 单一指标 不足以 入场
```

## 🚨 v4.2 新增铁律（基于 5/29 复盘）

### 铁律 4：**"不操作" 为默认选项**
```
助手 任何 操作建议前 必须 评估:
  • 不操作 是否 也是 选项?
  • 操作 的 收益 > 风险 + 成本?
  • 用户 是否 真的 问 操作?

if 操作收益 < 不操作:
  推荐 不操作
```

### 铁律 5：**已浮盈 +30% 小仓位特殊保护**
```
持仓 浮盈 > 30% + 仓位 < 5%:
  • 不主动建议 清仓 / 换仓
  • 即使 是 支线 / 短期跑输
  • Let winners run
  
触发减仓 的 唯一条件:
  • 触发 入场时设的 减仓位
  • 出现 重大利空 (业绩 / 监管)
  • 用户 主动提出
```

### 铁律 6：**同板块集中度上限**
```
加仓前必查:
  • 加仓后 同板块持仓 总市值
  • 加仓后 同涨跌相关 持仓 总市值
  
if 同板块 > 总持仓 60%:
  ⚠️ 警示 "过度集中"
  
if 同板块 > 总持仓 80%:
  🚨 拦截 "禁止加仓"
```

### 铁律 7：**主线/支线分类的适用边界**
```
主线识别引擎 (main_line_tracker):
  ✅ 适用: 新建仓 / 选板块
  ❌ 不适用:
    • 已浮盈 +30% 的仓位
    • 仓位 < 5% 的小仓
    • 7 天内已交易过的
  
"中天 = 支线" 不等于 "应该减仓"
"通富 = 主线" 不等于 "应该加仓"
还要看 个股状态 + 用户阶段
```

### 铁律 8：**操作频次限制**
```
同一只票:
  • 24 小时内 最多 1 次操作建议
  • 7 天内 最多 2 次操作建议
  
原因:
  • 频繁改建议 = 信号污染
  • 我 5/29 一天内 给中天 2 次建议 = 错误
  • 频繁操作 = 散户最大敌人
```

### 铁律 9：**卖飞防御机制**
```
if 卖出 后 24h 内 该票 涨 > 5%:
  • 触发 "卖飞反思"
  • 24h 禁止 对该票 新操作建议
  • 不允许 "卖飞 → 立刻 推剩余卖"
  • 等冷静 24h 后 再评估
```

### 铁律 10：**用户阶段识别**
```
根据用户 当前状态 调整建议风格:

🟢 建仓期 (现金 > 60%):
  → 积极推荐 操作
  → 主线龙头优先

🟡 持有期 (持仓 50-70% / 浮盈大):
  → 减少 操作推荐
  → 重点 是 "持有" 不是 "优化"

🔴 收割期 (已浮盈 + 接近顶部):
  → 主推 减仓 + 现金
  → 不再 加仓 主线

Michael 5/29 状态 = 持有期偏收割期
我误用 建仓期 思维 推 操作
```
